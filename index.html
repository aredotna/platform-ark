Use this tool to migrate your Tumblr blog to an Arena channel.
Tumblr Import

Go to https://www.tumblr.com/settings/apps and create a Tumblr application and fill out the information here.

Arena Export

Go to https://dev.are.na/oauth/applications and create an Arena application and fill out the information here.

Notes

• "Neue Post Format (NPF) Posts" format currently not supported.
